## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/tallinding-islamic-institution.git
cd tallinding-islamic-institution
```

2. Open `index.html` in your browser, or use a local server:
```bash
python -m http.server 8000
```

3. Visit `http://localhost:8000` in your browser
